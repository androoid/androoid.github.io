What is Androoid?

Androoid is an Spring Roo Addon Suite that provides developers all necessary tools to generate Android mobile applications quickly.

Android Java Applications

Generates your own Android mobile applications using Java language without programming knowledge.

Time saver

Save time generating your Android applications in less than 5 minutes!

Auto-generated code

Use simple commands provided by Androoid to generate your Android application automatically.

Easy to customize

Based on Spring Roo philosophy. Generated project uses AspectJ to maintain code automatically.

Simple User Interface

Provides a simple user interface that could be customized easily by developer
